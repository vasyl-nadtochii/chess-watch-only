# WatchOS-only Chess

Chess game developed specifically for Apple Watch. Inspired by [this video](https://youtu.be/U4ogK0MIzqk?si=0BPJ5YUqvyE9shag).

## Specifications

- Language: **Swift 5**
- UI Framework: **SwiftUI**
- Third-party libraries: **None**
- Min SDK version: **watchOS 8.0**
